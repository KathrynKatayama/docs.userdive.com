# CocoaPodsからインストール (推奨)

USERDIVE iOS SDKはCocoaPodsからインストールすることができます。

## Podfileの記述

`Podfile` を書きます。

```
pod 'Userdive-iOS-SDK', '~> 1.0'
```

## Podsのインストール

Podsをインストールします。

```
pod install
```

## 開発開始

ワークスペースを開きます。

```
open YourProject.xcworkspace
```
