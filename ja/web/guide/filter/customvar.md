# カスタム変数にセットできる内容

カスタム変数には下記の条件があります

- 3バイト以下の UTF-8 文字列 (`string`)
- 最大 5 つまで
- 最大 128 文字

**一意にユーザが特定できる文字列** の設定は非推奨とさせて頂いております。

- ID
- 氏名
- メールアドレス
