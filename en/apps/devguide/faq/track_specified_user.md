# 特定のユーザーだけトラッキングできますか

特定のユーザのみをトラッキングすることは出来ません。

アプリ上でカスタム変数を登録し、管理画面のフィルタ機能により絞り込むことで、
特定ユーザのデータのみを参照することができます。
