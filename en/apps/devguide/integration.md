# Install from CocoaPods (recommendation)

You can install USERDIVE iOS SDK from CocoaPods.

## How to write Podfile

Write `Podfile`.

```console
pod 'Userdive-iOS-SDK', '~> 1.0'
```

## Install Pods

Install Pods.

```console
pod install
```

## To start developing

Open workspace.

```console
open YourProject.xcworkspace
```
